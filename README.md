# MockUp - A Lightweight and Open Source Drawing and Animation software

***

## What is MockUp?:

MockUp is an open-source drawing and animation software. Lightweight with an emphasis on layout animation.

## What features does MockUp currently have?:

MockUp's feature list can be broken down into four main categories:
| **Illustration** | **Animation** | **User Specific** | **Canvas Specific** |
|:---:|:---:|:---:|---|
| Drawing with various colors, brushes, and brush sizes. | Creation, deletion, and management of animation **frames**. | Settings menu where user can change various app defaults. | Creation of multiple canvas to draw on. |
| Creation, deletion, and management of canvas **layers**. | Use of an animation layout template. | Customizable keybinds that allow quick actions by the user. | Selection of various preset sizes and custom sizes. |
| User can import their own custom brushes for use in the app. | Animation playback (& adjustable playback speed). | Choice of default UI and a customizable, modular UI setup. | Changeable paper color. |
| Undo/Redo functionality for drawing | Toggleable onion skins (& adjustable onion skin count). |  | Choice of creating an Illustration or Animation canvas.  |
|  |  |  |  |
